## 6. Идея, метод, алгоритм решения задачи

1) Релизовать вектор, динамический массив;
2) Реализовать функции инициализации таблицы, увелечение вместимости, добавления элемента;
3) Поиск элемент по ключу с помощью стандартного алгоритма бин поиска;
4) Реализовать три варианта присваивания ключей:
 - по возрастанию;
 - по убыванию;
 - случаные;
5) Сортировка ключей шейкерной сортировкой (пузырек в обе стороны);
6) Чтение песни из файла;
