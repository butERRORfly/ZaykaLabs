## 6. Идея, метод, алгоритм решения задачи

1) Разделим программу на 3 файла:
- pc_db.h - определеяем пространство имен, импортируем необходимые библиотеки, описываем структуры данных сущностей в БД;
- generate_db.c - генерация случайных записей в БД (кол-во записей выбираем сами, минмум 20);
- query_db.c - выполнение задания по варианту;

Общая идея:
С помощью файлового дескриптора наполняем БД записями (формат наполнения может быть любым), с помощью все того же
файлового дескриптора считываем сущности из БД и выполняем задания по варианту;
