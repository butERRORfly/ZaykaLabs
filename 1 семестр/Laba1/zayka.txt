Установить любую unix-систему, 8 лабороторок, занимаемся скорее всего дистанционно по понедельникам вечером.

1 лабороторная работа

who - какому пользователю принадлежит эта система;

pwd - путь до текущей директории (абсолютный путь от начала корневого каталога);

cd - перейти в директорию;

ls - показывает содержимое директории;

ls -l - подробная информация о файлах в директории;

man + команда - мануал о команде;

cd .. - выход из директории;

mkdir + название директории - создает директорию;

touch + название файла - создание файла;

nano + название файла - встроенный текстовый редактор;

rm + название файла - удаление файлов;

rmdir + название директории - удаление директории;

rm -r + название директории - рекурсивное удаление файлов в этой директории;

cat + название файла - выводит содержимое какого-то файла; 

echo - повторяет то, что ты ввел;

grep - фильтрация внутри файла;

clear - очистка терминала;

history - история команд;

ls -l > название файла - запись работы команды в файл;

ls -l >> название файла - дозапись в файл;

| - пайплайм - результат можем передать в другую команду; (ls -l | grep 0 > file.txt - в этом примере мы передаем результат работы команды ls в команду grep и записываем в файл file.txt только те строчки, которые содержат, например, 0);

sort + название файла - данные для своей работы берет из файла входного, сортирует лексикографически и результат выводит в терминал;

bach - запуст баш-скрипта;

chmod +x название баш-скрипта - сделать баш-скрипт исполняемый;
